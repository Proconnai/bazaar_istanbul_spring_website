# Bazaar Istanbul Spring Website

Official website for Bazaar Istanbul Spring - a premium halal meat and international grocery store in Spring, Texas.

## About

Bazaar Istanbul Spring offers authentic halal meats, Turkish delights, Mediterranean groceries, and Desi (Pakistani/Indian) products to the Spring and Woodlands community.

## Features

- Responsive single-page design
- Customer reviews section
- Product showcase
- Store hours and location with Google Maps
- Contact form
- Social media integration (Instagram, Facebook)
